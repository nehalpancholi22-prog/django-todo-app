# Django Todo App

A simple Todo List web application built using Django.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Clean Bootstrap UI

## Tech Stack
- Python
- Django
- HTML
- Bootstrap

## How to Run

1. Clone the repository

git clone https://github.com/nehalpancholi22-prog/django-todo-app.git

2. Go into the project

cd django-todo-app

3. Install Django

pip install django

4. Run the server

python manage.py runserver

5. Open in browser

http://127.0.0.1:8000
